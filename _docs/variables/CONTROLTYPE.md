Type of the current control.

Can be one of the following values:
* `BUTTON`
* `ICON`
* `LABEL`
* `LAYOUT` (Custom Gui)
* `PLAYBACKSTATUS`
* `PROGRESSBAR`
* `SLIDER`
* `TEXTAREA`
