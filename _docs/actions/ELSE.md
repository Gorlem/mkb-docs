The actions following this action will only be executed if no if-clause before evaluated to `true`.

Can be used with:
* `IF`
* `IFBEGINSWITH`
* `IFCONTAINS`
* `IFENDSWITH`
* `IFMATCHES`
* `ELSEIF`
